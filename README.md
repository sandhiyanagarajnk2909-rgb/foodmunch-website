# FoodMunch Website

FoodMunch is a responsive food ordering and restaurant landing page website built using HTML, CSS, and Bootstrap. The project provides an attractive and user-friendly interface for showcasing food services, menu items, delivery options, and promotional offers.

**Features**
Responsive Navigation Bar
Hero/Header Section
Why Choose Us Section
Explore Menu Section
Delivery & Payment Section
Gift Voucher Redemption using Bootstrap Modal
Follow Us Section
Footer Section
Mobile-Friendly Design using Bootstrap

**Technologies Used**
HTML5
CSS3
Bootstrap 5

**Project Purpose**
This project was developed to practice front-end web development concepts, including responsive design, Bootstrap components, layout creation, and interactive UI elements such as modals.
